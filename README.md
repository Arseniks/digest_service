# digest_service


## Копирование проекта
- Клонируйте проект с GitHub с помощью команды:
```commandline
git clone https://github.com/Arseniks/digest_service
```
- Перейдите в папку с проектом:
```commandline
cd digest_service
```
- Добавление виртуальной среды venv:
```commandline
python3 -m venv venv
```
